# SEO Submission Guide

## Google Search Console

### 1. Verify Ownership
- Go to: https://search.google.com/search-console
- Add property: `https://www.thepurplewings.org`
- Verification method: HTML tag (already in metadata) or DNS record

### 2. Submit Sitemap
```
https://www.thepurplewings.org/sitemap.xml
```

### 3. Request Indexing for Key Pages
- Homepage: https://www.thepurplewings.org
- Courses: https://www.thepurplewings.org/courses
- Blog: https://www.thepurplewings.org/blog
- About: https://www.thepurplewings.org/about
