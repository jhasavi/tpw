## HOW TO ADD RESOURCES TO LESSONS

### Step 1: Choose Resource Type
```
- "Calculator" (interactive tool)
- "Educational" (article, guide)
- "Course" (complete learning program)
- "Tool" (budget template, tracker)
- "Organization" (helpline, support)
- "Article" (blog post, research)
- "Video" (YouTube, tutorial)
- "Book" (written resource)
```

### Step 2: Verify Resource
- [ ] Link still works (test it)
- [ ] Content is current (published date)
- [ ] Source is reputable
- [ ] No paywalls (or clearly marked if paid)
- [ ] Mobile-friendly
- [ ] No major ads/spam

### Step 3: Add to Lesson JSON
```json
{
  "resources": [
    {
      "title": "Resource Name",
      "type": "Calculator|Educational|Course|Tool|Organization|Article|Video|Book",
      "url": "https://...",
      "description": "One-sentence description of what it does for this lesson topic",
      "womenSpecific": true/false
    }
  ]
}
```

### Step 4: Quality Standards
**Every resource must:**
- ✅ Be specifically relevant to that lesson topic
- ✅ Work for beginners (not too technical)
- ✅ Have clear instructions on how to use
- ✅ Be free or clearly mark cost
- ✅ Have been tested in last 6 months

**Example: For "Budgeting" Lesson**
```json
{
  "title": "Google Sheets Budget Template",
  "type": "Tool",
  "url": "https://docs.google.com/spreadsheets/u/0/templates",
  "description": "Free, customizable budget spreadsheet you can copy and modify. Tracks income, expenses, and savings goals monthly.",
  "womenSpecific": false
}
```

## USING THIS LIBRARY

### For Content Writers
1. Use this list when creating lesson content
2. Link 3-5 most relevant resources per lesson
3. Test all links before publishing
4. Verify resources quarterly for broken links

### For Web Development
1. Create `/resources` page with full library
2. Link lessons → relevant resources
3. Add resource type filters (calculators vs articles, etc)
4. Track clicks to understand what women use most

### For Quality Assurance
1. Monthly check: Test 20% of links
2. Quarterly: Add new resources, remove dead links
3. Annually: Audit entire library for relevance

### For Women Using Platform
1. See related resources as you complete lesson
2. Bookmark favorites in personal resource library
3. Filter by type (calculators, organizations, courses)
4. Export useful links to share with friends
